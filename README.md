# Copy as FFUF 

Copy request as ffuf command.

### requires

- [ffuf](https://github.com/ffuf/ffuf)
- [pencode](https://github.com/ffuf/pencode)

## Menu Items

- Copy as FFUF ( Minimal Headers )
- Copy as FFUF (Full Headers )
- Copy as FFUF (Encoded Wordlist; requires pencode )
